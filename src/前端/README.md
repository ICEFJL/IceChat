## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### 桌面端打包

```
npm run electron:build
```

打包结果位于release文件夹下

参考：[基于Electron24+Vite4+Vue3搭建桌面端应用 - xiaoyan2017 - 博客园](https://www.cnblogs.com/xiaoyan2017/p/17436076.html)
